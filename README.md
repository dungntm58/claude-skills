# claude-skills

Personal Claude Code skills, installable as a plugin.

## Install

```
/plugin marketplace add <github-user>/claude-skills
/plugin install claude-skills
```

## Skills

- **chaining-openspec-superpowers** — orchestrates openspec (proposal + specs) with superpowers (design + plan + subagent dispatch) end-to-end.
- **nano-banana-sprites** — generate 2D pixel/retro game sprites (characters, sheets, animation frames, tilesets) via Google's Nano Banana image API, with pixel-grid + palette post-processing.
